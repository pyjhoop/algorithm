## 스택
- 스택은 쌓다라는 의미로 위로 데이터를 쌓는 이미지를 생각하면된다.
- 쌓기에 LIFO인 후입 선출의 특징을 가지고 있고
- 데이터는 단방향으로 이동이 가능하다.

## 스택 주요 메서드
- stack.push() : 데이터를 삽입
- stack.pop() : 데이터 추출 후 반환
- stack.peek() : 데이터 반환
- stack.empty() : 비어있으면 true, 차있으면 false
- stack.search() : 해당 값이 있으면 위치 반환 여러개 있으면 마지막 위치 반환
- stack.contains() : 해당 값이 있으면 true
- stack.size() : 스택의 크기를 리턴
